# Banana Token (BANANA)

**Sample commodity token on Pentagon Chain**

---

## Overview

BANANA is a sample ERC-20 commodity token demonstrating the Pentagon Chain token standard. Used for testing, faucets, and example integrations.

---

## Token Details

| Property | Value |
|----------|-------|
| Standard | ERC-20 |
| Decimals | 18 |
| Network | Pentagon Chain (3344) |

---

## Functions

| Function | Description |
|----------|-------------|
| `transfer(to, amount)` | Standard ERC-20 transfer |
| `approve(spender, amount)` | Approve spending |
| `mint(to, amount)` | Mint new tokens (admin) |
| `burn(amount)` | Burn tokens |
