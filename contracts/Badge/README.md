# Badge Contracts

**Soulbound achievement badges for the Pentagon ecosystem**

---

## Overview

Generic soulbound badge system for achievements, milestones, and certifications across Pentagon Chain applications.

---

## Functions

| Function | Description |
|----------|-------------|
| `mint(to, badgeId)` | Award badge to address |
| `balanceOf(owner, id)` | Check if address holds badge |
| `uri(id)` | Get metadata URI for badge |

---

## Soulbound

All badges are non-transferable. Once earned, they stay with the wallet forever.
