# Proof of Work SBT (2025)

**Soulbound tokens proving ecosystem participation**

---

## Overview

POW SBT rewards active participants in the Pentagon ecosystem. These non-transferable tokens prove your contribution history and unlock benefits.

---

## Token Details

| Property | Value |
|----------|-------|
| Standard | ERC-1155 Soulbound |
| Transferable | No |
| Network | Pentagon Chain (3344) |

---

## Earning POW

Tokens are earned through:
- Game participation
- Community contributions
- Event attendance
- Development contributions
- Bug bounties

- VIP access tiers
- Reward multipliers
- Governance weight
- Exclusive drops
- Early access
