## API Endpoints

| Endpoint | Description |
|----------|-------------|
| `/api/marketplace/list` | List NFT for sale |
| `/api/marketplace/buy` | Buy listed NFT |
| `/api/marketplace/offer` | Make offer |
| `/api/marketplace/cancel` | Cancel listing |
| `/api/marketplace/collections` | Get collections |
| `/api/marketplace/activity` | Get activity feed |
